# Keep It Simple Stupid Code of Conduct
* We are all adults, capable of making adult decisions.
* We accept anyone's contributions.
* USE YOUR COMMON SENSE. If it shouldn't be typed, don't type it.
* The code is our focus, not our personal politics. Please refrain from bringing it here.
* TREAT OTHERS HOW YOU WANT TO BE TREATED. Always assume good intentions and be charitable.
* Any attempts to weaponize this CoC will result in an immediate ban.
* Beyond that, nothing else matters.

This CoC is based on [No Code of Conduct](https://github.com/domgetter/NCoC). 
For more information, please visit the [Keep It Simple Stupid Code of Conduct](https://github.com/xseadgdc/KISS-CoC) homepage.
